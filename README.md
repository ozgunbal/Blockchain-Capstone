# Udacity Blockchain Capstone

The capstone will build upon the knowledge you have gained in the course in order to build a decentralized housing product.

## Changes from the boilerplate
- Used dependency versions:
  - @truffle/hdwallet-provider: ^2.0.11
  - truffle ^5.5.13
  - @openzeppelin/contracts: ^4.6.0
  - Solidity: ^0.8.0
- Oraclize is removed. For string concatenation, `string(abi.encodePacked(_baseTokenURI, Strings.toString(tokenId)))` is used with the help of OpenZeppelin's `Strings.sol`
- To generate verifier and proofs, Remix IDE is used.

## How to
 - To run tests `cd eth-contracts` then run `truffle test` or `npx truffle test`(if it's not globally installed)
 - To deploy rinkeby `npm run deploy` is runned in the root directory
 - For minting 10 tokens with 10 different proof, `node mint.js` is run at the `eth-contracts` folder
